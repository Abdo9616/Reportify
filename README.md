- Install: npm install

- Copy .env.example -> .env and fill values

- Run: `node index.js`

- Commands are registered to GUILD_ID if provided (fast). Remove GUILD_ID to register globally (may take up to 1 hour).

- You can invite Reportify directly to your server [here](https://discord.com/oauth2/authorize?client_id=1429509460274974780&permissions=68608&integration_type=0&scope=bot+applications.commands).
